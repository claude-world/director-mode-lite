# CLAUDE.md Template Generator

Generate a customized CLAUDE.md template based on detected project type.

## Detection

First, detect project characteristics:

1. **Language**: Check for package.json, requirements.txt, Cargo.toml, go.mod
2. **Framework**: Parse dependencies for React, Vue, FastAPI, Express, etc.
3. **Package Manager**: npm, pnpm, yarn, pip, cargo
4. **Test Framework**: jest, pytest, cargo test
5. **Existing Patterns**: Check for .eslintrc, .prettierrc, tsconfig.json
